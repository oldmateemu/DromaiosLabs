# Decisions

## 2026-05-29: Thin Custom Cockpit

Dromaios Cockpit is a thin custom operating layer, not a replacement for every specialist tool. It stores actions, reviews, links, drafts, risks, decisions, and automation state while linking out to tools like Xero, Airwallex, Lawpath, Skool, GitHub, and n8n/Activepieces.

## 2026-05-29: Local/VPN First

The first runnable version is local/VPN-first. Public-domain hardening, OAuth, multi-tenant access, and external monitoring are deferred.

## 2026-05-29: Ollama First

Ollama is the default assistant provider for private and routine company-memory work. Cloud AI is optional and gated by privacy rules.
